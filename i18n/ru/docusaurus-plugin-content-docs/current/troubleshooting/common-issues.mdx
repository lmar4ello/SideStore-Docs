## Частые ошибки с SideStore

### AFC Connection Failure
#### Причины возникновения
Проблемы с подключением или неправильно-привязанный pairing файл.
#### Решение
1. Попробуйте переподключиться.
2. Сгенирируйте новый файл сопряжения (pairing file) используя JitterbugPair и вставьте его через меню с настройками. Посмотрите нашу [инструкции с файлом сопряжения](/docs/installation/pairing-file) для более подробных деталей.

### No WiFi or VPN Connection (Нет подключения к WiFi или VPN-у)
#### Причина возникновения
Проблема с подключением к сервису minimuxer.
#### Решение
1. Если используете WireGuard как прокси, перейдите на StosVPN.
2. Попробуйте снова (спамить на кнопку обычно помогает) 
3. Перезапустите SideStore
4. Перезапустите StosVPN
5. Сгенирируйте новый файл сопряжения (pairing file) используя JitterbugPair и вставьте его через меню с настройками. Посмотрите нашу [инструкции с файлом сопряжения](/docs/installation/pairing-file) для более подробных деталей.

### Sidestore сбрасывается на половине установки
#### Причина возникновения
Проблемы с кешом.
#### Решение
1. Перезапустите SideStore
2. Очистите кеш
3. Поменяйте сервер Anisette
4. Сбросьте adi.pb
5. Выйдите из SideStore и зайдите обратно
6. Попробуйте используйте WireGuard
7. Перезапустите устройство.Restart device
8. Сгенирируйте другой файл сопряжения (pairing file)
9. Переустановите SideStore

### Не получается запустить SideStore
#### Причина возникновения
Много возможных причин.
#### Решение
1. Убедитесь что вы установили SideStore через Altserver.
2. Переустановите SideStore

Если решения сверху не работают, сделайте следующее:

1. Удалите Sidestore и все приложения которые вы установили с помощью SideStore.
2. Переустановите SideStore (да, опять).
3. Выберите файл сопряжения и войдите. (Вылезет промт чтобы обновить SideStore, выберете "нет")
4. Загрузите файл Sidestore.ipa на ваше устройство.
5. Импортируйте файл SideStore.ipa в SideStore также, как вы бы скачивали любое другое приложение.
6. Это разрешит SideStore сделать чистое обновление, и, надеюсь, исправит проблему.
